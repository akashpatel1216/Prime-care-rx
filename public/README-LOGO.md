# Logo Files

Please add your logo files to this directory:

## Required Logo Files:

1. **logo.png** or **logo.svg** - Main logo (recommended size: 200x60px or similar aspect ratio)
2. **logo-white.png** or **logo-white.svg** - White version for dark backgrounds
3. **favicon.ico** - Favicon for browser tab (32x32px)

## Recommended Formats:

- **SVG**: Best for scalability and quality (preferred)
- **PNG**: Good alternative with transparent background
- Dimensions: Width 150-250px, Height 40-80px (maintain aspect ratio)

## Current Usage:

The logo will be used in:
- Navigation header
- Footer
- Mobile menu
- As favicon in browser tab

After adding your logo files, the website will automatically use them.
